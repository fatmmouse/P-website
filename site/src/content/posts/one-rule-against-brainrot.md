最近三四天状态特别差，什么都不想学，就无脑刷视频。今天试着用 Claude 学了一篇 Kimi 最新论文，结果聊了一个多小时，居然进入心流了。

秘诀就是一条提示词规则：

**核心规则：**
"不要一次讲太多，一个概念确认理解了再往下走。通过提问引导我思考，一次一个问题。比喻要精确，不确定的内容先搜索验证再讲，不要编。所有内容讲完后，主动要求我用自己的话复述一遍。"

**实际效果：**
· 它每次只讲一个点，然后反问我"你觉得为什么？"——逼我自己推
· 我说"不懂"它就停下来换角度，不会硬往下灌
· 我说"你这个没依据吧"它就真的去搜论文验证
· 最后让我自己用嘴把整条逻辑链复述一遍

一个多小时，从零走到完整理解 Agent Swarm 的架构、训练机制、奖励函数设计，全程我都在思考和表达。

**和普通"问 AI"的区别：**
普通方式是你问一个问题，AI 哗啦啦给你一整篇。你看完觉得"嗯懂了"，但其实三分钟后就忘了。这个方式反过来——**它问你，你来答。**

**另外，用 Chat 界面而不是 Code 界面：**
这点很重要。Chat 界面文字是一个个流式蹦出来的，你的注意力被持续牵引着，不会断。但如果用 Claude Code 那种界面，它一跑几十秒甚至几分钟的等待，你的注意力一脱轨就去拿手机了，心流直接碎掉。人的注意力是脆弱的，**环境设计稍微不对，"在场"就断了。**

---

但今天最大的收获不是学到了什么知识，也不是什么提示词技巧。

是我突然意识到，刷视频和真正学习的区别，不是内容质量的区别。是**你有没有在说话**的区别。

刷视频的时候你只是在看，你他妈就是被喂的，被算法牵着走，笑了也是条件反射，脑子根本不转。但你开口说话的时候不一样——你得组织语言，做判断，承认"这个我懂，那个我不懂"。

你光看，你就只是被输入。你说出来，你才是一个在场的人。

这几天我真切体会到的是：**一个是你消失在内容里，一个是你在内容里找到自己。**
